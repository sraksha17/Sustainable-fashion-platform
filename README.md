# Sustainable Fashion Recycling Platform

## Description
A web platform to donate, resell, and recycle clothes, helping reduce fashion waste.

## Features
- User authentication (login/signup)
- Product listing and browsing
- Donation system
- Payment integration

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Flask (Python)
- Database: MongoDB

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the application:
   python app.py
4. Open browser:
   http://127.0.0.1:5000
